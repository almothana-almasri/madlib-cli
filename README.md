## Setup

1. Create a virtual environment (optional):

```bash
python3 -m venv .venv
source .venv/bin/activate
```
2. Install required packages:

```bash
pip install -r requirements.txt
```

## How to run the game:

```bash
python madlib_cli/madlib.py
```

## Tests

```bash
pytest tests
```
